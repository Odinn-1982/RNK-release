## Reporting Issues

Found a bug or have a feature request? Please report it on our [GitHub Issues](https://github.com/Odinn-1982/RNK/issues) page.

When reporting bugs, please include:
1. Foundry VTT version
2. System version
3. Browser and version
4. Steps to reproduce
5. Expected vs actual behavior
6. Any error messages from the console (F12)
