# Screenshots Directory

Place your system screenshots here for use in documentation.

To complete the README documentation, please add screenshots with these names:

| Filename | Description |
|----------|-------------|
| `character-sheet.png` | Full character sheet overview |
| `skills-tab.png` | Skills tab showing skill organization |
| `combat-tab.png` | Combat tab with weapons and armor |

| `roll-dialog.png` | Dice roll dialog |

## How to Take Screenshots

### In Foundry VTT

1. Open the character sheet or feature you want to capture
2. Press `F12` to open developer tools
3. Press `Ctrl+Shift+P` (Windows) or `Cmd+Shift+P` (Mac)
4. Type "screenshot" and select "Capture screenshot"
5. Save to this folder with the appropriate name

### Alternative Methods

- **Windows:** `Win + Shift + S` for Snipping Tool
- **macOS:** `Cmd + Shift + 4` for selection capture
- **Linux:** Use your screenshot tool of choice

## Recommended Settings

- **Resolution:** 1920x1080 or higher
- **Format:** PNG for best quality
- **Size:** Crop to show only the relevant UI element
- **Window:** Use a clean world with minimal custom styling

## Tips

1. Use a character with filled-in data for more appealing screenshots
2. Ensure no personal information is visible
3. Use the default system styling (no custom CSS)
4. Take screenshots with the Foundry VTT UI chrome visible if showing the full sheet
